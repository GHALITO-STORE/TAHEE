# TAHEE - Printful Store

A modern e-commerce site powered by Printful and built with Next.js.

## Features
- Product showcase from Printful
- Shopping cart integration
- Order management
- Responsive design

## Getting Started

### Prerequisites
- Node.js 18+
- Printful API credentials

### Installation

```bash
npm install
```

### Environment Setup

Create a `.env.local` file:

```
NEXT_PUBLIC_PRINTFUL_API_KEY=your_printful_api_key
PRINTFUL_API_SECRET=your_printful_api_secret
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build for Production

```bash
npm run build
npm start
```
